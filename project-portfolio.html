Project Overview

A premium, production-ready developer portfolio designed with a cyberpunk glassmorphism aesthetic. This single-page application showcases projects, skills, and achievements with smooth animations, live GitHub API integration, and a responsive design that looks stunning on every device. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks required.

Design Goals

The primary goal was to create a portfolio that immediately communicates professionalism and technical competence — something a recruiter opens and instantly recognizes as the work of a serious developer, not a student project. The design draws inspiration from cyberpunk aesthetics with deep navy backgrounds, neon cyan accents, glassmorphism cards, and smooth micro-animations that make every interaction feel premium.

Key Features

Responsive Design

Fully responsive layout that adapts seamlessly from mobile phones to ultra-wide desktop displays using CSS Grid and Flexbox.

Glassmorphism UI

Premium frosted glass cards with backdrop blur effects, translucent backgrounds, and subtle border highlights.

Cyberpunk Theme

Deep navy (#050816) background with vibrant neon cyan accents, gradient glowing buttons, and particle background effects.

Project Showcase

Dedicated case-study pages for each project with detailed problem statements, solutions, tech stacks, and results.

Contact Form

Production-ready contact form integrated with FormSubmit for direct email delivery with spam protection.

Resume Preview

Integrated resume preview modal with embedded PDF viewer and one-click download functionality.

Challenges & Solutions

Challenge

Cross-Browser Glassmorphism

The backdrop-filter CSS property used for glassmorphism has inconsistent support across browsers, particularly older versions of Firefox.

Solution

Progressive Enhancement

Implemented progressive enhancement with solid fallback backgrounds and used -webkit-backdrop-filter alongside standard properties for maximum compatibility.

Challenge

Performance with Particles

The Particles.js background animation caused frame drops on lower-end mobile devices, impacting scroll smoothness.

Solution

Reduced Particle Count

Reduced particle density on mobile via responsive configuration and enabled hardware-accelerated CSS transforms for smooth animations.
