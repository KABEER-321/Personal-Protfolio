Project Overview

SmartCampus ERP Portal is a comprehensive full-stack academic management platform designed to digitize and streamline every aspect of campus operations. Built with Django and enhanced by Google's Gemini AI, it provides role-based dashboards for students, staff, parents, and administrators — all powered by intelligent automation and real-time data insights.

Problem Statement

Traditional academic management relies on fragmented systems — paper-based attendance registers, disconnected grade spreadsheets, manual notification processes, and zero visibility for parents. This creates administrative overhead, data inconsistencies, delayed communication, and an overall poor experience for every stakeholder in the academic ecosystem.

Solution Approach

SmartCampus consolidates all academic operations into a single, AI-enhanced web platform. By implementing role-based access control with Django's authentication system and integrating Google's Gemini API with Retrieval Augmented Generation (RAG), the portal delivers personalized dashboards, automated attendance tracking, intelligent grade analytics, and an AI chatbot that can answer questions about academic policies, schedules, and student performance using the institution's own knowledge base.

Challenges & Solutions

Challenge

Role-Based Access Control

Managing four different user roles (Student, Staff, Parent, Admin) with distinct permissions and data visibility was architecturally complex.

Solution

Django Groups & Middleware

Leveraged Django's built-in Groups and Permissions framework with custom middleware to enforce role-based routing and data access at the view level.

Challenge

AI Context Accuracy

The AI chatbot initially hallucinated answers when asked about specific institutional policies or student data not in its training data.

Solution

RAG Pipeline

Implemented Retrieval Augmented Generation (RAG) to ground the Gemini API's responses in the institution's actual documents and database records, dramatically improving accuracy.
